# SpamDetection
An end-to-end project for email and sms spam detection system

# How to build the docker image and run the container
1. docker build -t spamDetection .
2. docker run -p 8501:8501 spamDetection

# Evaluation Metrices:
1. Accuracy
2. Precision
3. Recall
4. F1 Score
5. Confusion Matrix
